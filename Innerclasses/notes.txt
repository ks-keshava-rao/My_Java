1. Types :- Nested inner class 
            local inner class
            anonymous inner class
            static inner class 
2. Inner class can access all the members of outer class directly but outclass cannot access inner class object direclty .
3. By creating the object on inner class inside outer class members can be accesses  
4. Inner class file will be created by outer$inner.java 
5. Local inner class are classes that are inside the local method , this is useful when you want to write a class that is inheriting or implementing from interface 
   from existing class . 
6. Anonymous  class :- Can be defined at the  time of creating the object , useful for interfaces and abstract classes
7. Static inner class :-  object of static inner class can be directly created without creting the object of outer class . static inner class can only access the 
   static members of outer class .
